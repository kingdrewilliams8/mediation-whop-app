# Soothing Sounds Audio Files

Place your soothing sounds audio files in this folder with the following naming convention:

- `ocean-waves.mp3` - Ocean waves sound
- `rain-forest.mp3` - Rain forest sound  
- `evening-crickets.mp3` - Evening crickets sound
- `thunder-storm.mp3` - Thunder storm sound
- `morning-birds.mp3` - Morning birds sound
- `babbling-brook.mp3` - Babbling brook sound

All files should be in MP3 format.
